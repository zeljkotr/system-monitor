# ⚡ System Monitor - Web Dashboard

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Flask](https://img.shields.io/badge/Flask-2.x-green)
![Platform](https://img.shields.io/badge/Platform-Linux-orange)

Live web dashboard za monitoring sistema u realnom vremenu.

## ✅ Šta prikazuje
- CPU korišćenje sa upozorenjem
- RAM korišćenje i ukupna memorija
- Disk korišćenje i slobodan prostor
- Mrežni saobraćaj (TX/RX)

## 🛠️ Tehnologije
- Python 3
- Flask
- psutil

## 🚀 Pokretanje
pip3 install flask psutil
python3 app.py

## 🌐 Pristup
http://IP_SERVERA:5000
